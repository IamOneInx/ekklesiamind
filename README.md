# ekklēsia Ministry Driver

A mobile-first prototype for Anabaptist Ministry Drivers serving neighbors with appointment trips.

This first version demonstrates:

- Apt trip scheduling
- Start/advance/complete trip flow
- Time, odometer, mileage, and donation record entry
- EMD member sign-up and complete driver portfolio concept
- Member-only driver map opt-in for neighborhood dispatching
- Driver-set suggested donation settings
- Taxi-style suggested donation formula with optional extra fees
- Printable trip receipt and device total preview
- Stewardship summary cards
- Service-first wording that avoids commercial taxi language

## Run locally

```bash
npm install
npm run dev
```

## Verify

```bash
npm test
npm run lint
npm run build
```
